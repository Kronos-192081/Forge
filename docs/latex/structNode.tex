\doxysubsection{Detailed Description}
Represents a node in a dependency graph with associated target data. 

This struct encapsulates a node\textquotesingle{}s name and its associated target data. It provides comparison operators for equality, inequality, and ordering, as well as a friend function for outputting the node\textquotesingle{}s name to an output stream.
