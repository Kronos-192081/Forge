\doxysubsection{Detailed Description}
Specialization of the {\ttfamily std\+::hash} template for the {\ttfamily \doxylink{structNode}{Node}} struct. 

This specialization allows {\ttfamily \doxylink{structNode}{Node}} objects to be used as keys in unordered containers such as {\ttfamily std\+::unordered\+\_\+map} or {\ttfamily std\+::unordered\+\_\+set}. It computes the hash value for a {\ttfamily \doxylink{structNode}{Node}} based on its {\ttfamily name} attribute.
