   \hypertarget{index_autotoc_md0}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md0}
Forge is a {\bfseries{modern}} and {\bfseries{user-\/friendly build system}} written in {\bfseries{C++20}}. It is designed to simplify and optimize the build process for projects of any size. With its powerful features and intuitive design, Forge is the perfect tool for developers looking for a flexible and efficient build system.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Decomposition of Build Configurations}}\+: ~\newline
 Forge allows you to split build configurations into multiple reusable files, making it easier to manage complex projects.
\item {\bfseries{Python-\/\+Like Imports}}\+: ~\newline
 Supports Python-\/like imports, optimizing the parsing and processing phases of configurations.
\item {\bfseries{HTML Build Results}}\+: ~\newline
 Generates {\bfseries{HTML outputs}} of the build results, making them easy to read and understand.
\item {\bfseries{Caching}}\+: ~\newline
 Implements caching mechanisms to avoid redundant builds, improving performance.
\item {\bfseries{Parallel Processing}}\+: ~\newline
 Supports parallel execution of tasks, significantly speeding up the build process.
\item {\bfseries{Ease of Use}}\+: ~\newline
 Designed to be simple and intuitive, making it easy to learn and use for developers of all skill levels.
\item {\bfseries{Flexibility}}\+: ~\newline
 A powerful and flexible system that can be adapted to any project.
\end{DoxyItemize}
